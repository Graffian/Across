Privacy Policy

Last updated: May 2026

Data Collection

Across does not collect, transmit, or share any personal data with third parties. All processing is performed locally on your machine.

What Data is Stored

  • Tab content: Text extracted from web pages you visit is processed into semantic chunks and stored in a local PostgreSQL database you operate.
  • Embeddings: Vector representations of page content are generated via Jina AI's API and stored locally in pgvector.
  • Chat history: Your chat sessions with the AI are stored locally in IndexedDB within your browser.
  • Tab metadata: URLs, titles, and favicons are stored locally in your PostgreSQL database.

Third-Party Services

Across sends page content to the following services (both require API keys you configure):

  • Jina AI (embeddings.jina.ai) — for generating semantic embeddings of page content.
  • Groq (api.groq.com) — for generating AI chat responses based on indexed content.

You are responsible for the API keys used, and data sent to these providers is subject to their respective privacy policies.

Network Requests

All network requests are sent to the backend server you configure (default: http://localhost:3001). No data is sent to Across-operated servers.
